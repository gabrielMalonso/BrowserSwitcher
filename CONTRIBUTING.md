# Contributing to BrowserSwitcher

Thanks for your interest in contributing! Here's how to get started.

## How to contribute

1. **Fork** the repository
2. **Create a branch** from `main` for your change (`git checkout -b my-feature`)
3. **Make your changes** and test them locally
4. **Commit** with a clear message describing what you changed and why
5. **Push** to your fork and open a **Pull Request** against `main`

## Pull request guidelines

- Keep PRs focused -- one feature or fix per PR
- Include a clear description of what the PR does and why
- Make sure the project builds with `swift build`
- Follow the existing code style (SwiftUI + AppKit patterns)

## Reporting issues

- Use [GitHub Issues](https://github.com/gabrielMalonso/BrowserSwitcher/issues) to report bugs or suggest features
- Include your macOS version and steps to reproduce

## Code of conduct

Be respectful and constructive. This is a small project -- keep discussions friendly and focused.
